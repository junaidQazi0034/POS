Point of Sale (POS) system

The Point of Sale (POS) system is a web-based application built using Python and the Flask web framework. Its primary purpose is to facilitate sales and inventory management for small to mediumsized businesses. This system allows users to manage products, process sales, generate receipts, and monitor inventory. It's designed to enhance the efficiency and accuracy of sales transactions.
